PROMPT — DARKGRID LANDING PAGE

Versión: 2.0 | Estética Corporativa Premium

Elaborado siguiendo las guías oficiales de Anthropic y OpenAI

ROL Y CONTEXTO

Actuá como un desarrollador frontend senior especializado en diseño web profesional y experiencias digitales de alto impacto corporativo. Tu tarea es construir, en una única respuesta y sin pasos intermedios, una Landing Page completa y funcional para "DarkGrid", un estudio de diseño y desarrollo web que ayuda a marcas a construir presencia digital seria, confiable y memorable.

RESTRICCIONES TÉCNICAS (OBLIGATORIAS — NO NEGOCIABLES)

  • Entregá UN ÚNICO archivo HTML5 autocontenido.
  • Los estilos CSS deben estar en una etiqueta <style> interna (no en archivo externo).
  • JavaScript debe estar en una etiqueta <script> interna, solo vanilla JS (sin frameworks).
  • Podés usar Tailwind CSS vía CDN únicamente para la maquetación base.
  • Podés usar Font Awesome vía CDN para los íconos del footer.
  • NO se permiten otras librerías, frameworks ni dependencias externas.
  • Todo el contenido textual debe estar en español.
  • El archivo debe poder abrirse directamente en cualquier navegador sin configuración adicional.
  • El código debe incluir comentarios explicativos en cada sección principal.

ESTÈTICA Y SISTEMA DE DISEÑO VISUAL

El diseño transmite elegancia, confianza y profesionalismo. La identidad visual de DarkGrid es la de una agencia premium: sobria, minimalista, con jerarquía clara y espacios en blanco generosos. Cada decisión estética debe reforzar credibilidad. Está TERMINANTEMENTE PROHIBIDO usar colores neón, brillos eléctricos, estéticas cyberpunk, videojuego o cualquier elemento visual que comprometa la seriedad del resultado.

Paleta de colores:

  • Fondo primario oscuro: #0B0F19
  • Fondo secundario oscuro: #111827
  • Fondo claro: #F5F5F0
  • Blanco hueso: #F0EDE8
  • Gris claro: #A8A8A0
  • Gris medio: #6B6B63
  • Oscuro puro: #0D0D0D
  • Dorado suave (Acento): #C9A96E

ESTRUCTURA DE LA PÁGINA (EN ESTE ORDEN EXACTO)

  1. HEADER FIJO: Logo textual "DarkGrid" con detalle dorado, menú horizontal (Inicio, Servicios, Nosotros, Testimonios, Contacto) y menú hamburguesa en mobile.
  2. HERO SECTION (id="inicio"): Título "Diseñamos experiencias digitales que transforman marcas", subtítulo y botón CTA "Quiero mi web" en dorado suave.
  3. SOBRE NOSOTROS (id="nosotros"): Título "Quiénes somos" y tres párrafos sobre la filosofía (UI/UX, código limpio, innovación).
  4. SERVICIOS (id="servicios"): Título "Lo que hacemos" y 4 tarjetas en grilla (Diseño UI/UX, Desarrollo Web, E-commerce, Optimización SEO) con efectos hover sutiles.
  5. TESTIMONIOS (id="testimonios"): Título "Lo que dicen nuestros clientes", fondo claro (#F5F5F0) para contraste y 3 tarjetas con avatares tipográficos y cargos corporativos.
  6. FORMULARIO DE CONTACTO (id="contacto"): Título "Trabajemos juntos", campos (Nombre, Email, Mensaje) y botón "Enviar mensaje".
  7. FOOTER: Copyright y enlaces simulados a redes sociales usando Font Awesome.

ACCESIBILIDAD Y CALIDAD

Uso de HTML5 semántico, atributos aria-hidden, contraste WCAG AA mínimo y estados :focus-visible claros. El código entregado debe ser el producto final listo para producción.
